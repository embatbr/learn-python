Introdução

Se você é um programador recente, talvez conheça apenas sua primeira linguagem (provavelmente Java) e nunca viu um goto, então a resposta é fácil: para ter mais opções. Mas você também pode ser jurássico, dos tempos do COBOL e do BASIC. Neste caso minha melhor resposta é: para ter mais opções.

Não importa quantos anos de experiência você tenha, ou qual foi sua primeira linguagem, Python lhe mostrará uma nova maneira de programar. A maneira divertida!

A seguir darei diversos motivos para aprender Python.


Python é amiga do programador

Com uma sintaxe simples e intuitiva, Python tem um foco na legibilidade do código. Além disso, não é preciso criar classes com um método estático nem importar bibliotecas apenas para escrever um simples Hello World. Abaixo uma breve comparação entre Java, C e Python:

Java: 

public class HelloWorld
{
    public static void main(String[] args)
    {
        System.out.println("Hello World!");
    }
}
$ javac HelloWorld.java
$ java HelloWorld
Hello World!
$

C: 

#include <stdio.h>

int main(void)
{
    printf("Hello World!\n");
    return 0;
}
$ gcc helloworld.c -o hello.exe
$ ./hello.exe
Hello World!
$

Python: 

print("Hello World!")
$ python3.2 helloWorld.py
Hello World!
$

Viu como é fácil? Se eu quero apenas escrever "Hello World!" na tela, eu só deveria precisar enviar ao computador o comando escreva "Hello World!", não ter que criar uma estrutura de objeto, nem chamar uma biblioteca padrão como stdio (Standard IO - Entrada e Saída Padrão). O resto o sistema operacional deve abstrair. Isto leva a...


Abstrai o desnecessário

Python possui algumas características que muitos programadores Java ainda esperam: herança múltipla, valor default num parâmetro de função, dicionários e etc. Em java, se você quiser que seu método possua um valor default em um parametro, é necessário duplicá-lo chamando o original.

Exemplo: 

public class CriarFrase
{
    ...
    public static String criarFrase(String[] palavras)
    {
        return this.criarFrase(palavras, ' ');
    }

    public static String criarFrase(String[] palavras, char separador)
    {
        String frase = palavras[0];
        for(int i = 1; i < palavras.length; i++)
        {
            frase = frase + separador + palavras[i];
        }

        return frase;
    }
    ...
}

As chamadas dos métodos com e sem o separador: 

    ...
    String[] palavras = {"Criando", "uma", "frase"};
    System.out.println(CriarFrase.criarFrase(palavras));
    System.out.println(CriarFrase.criarFrase(palavras, ' '));
    System.out.println(CriarFrase.criarFrase(palavras, '-'));
    ...

Possuem saídas: 

Criando uma frase
Criando uma frase
Criando-uma-frase

Já em Python basta fazer: 

def criarFrase(palavras, separador=' '):
    frase = palavras[0]
    for i in range(1, len(palavras)):
        frase = frase + separador + palavras[i]

    return frase

Com chamadas: 

palavras = ['Criando', 'uma', 'frase']
print(criarFrase(palavras))
print(criarFrase(palavras, ' '))
print(criarFrase(palavras, '-'))

E saídas: 

Criando uma frase
Criando uma frase
Criando-uma-frase

No código Java, para ter mais de um valor default é necessário criar todas as combinações desejadas. Isto leva a um mal aproveitamento do esforço do programador.


É modular

Python foi pensada para ser o mais modular possível. Suas estruturas não são divididas em classes (pois não é apenas OO), mas em módulos. Cada arquivo .py é um módulo, podendo conter diversas variáveis, classes e funções. Os módulos também podem ser agrupados dentro de pacotes (semelhante ao que ocorre em Java). Contudo caso o programador decida, por exemplo, modificar o nome de uma função, não é necessário alterar todas as chamadas. Basta apenas exportar com um "nome de guerra".


Dá para ganhar dinheiro

Embora não seja uma linguagem nova (sua primeira versão data de 1991), Python está experimentando um crescimento acelerado em sua base de desenvolvedores. Com uma evolução constante, a linguagem ganha cada dia mais adeptos. Por seu desenvolvimento aberto, qualquer desenvolvedor pode ajudar a melhorar a linguagem ou criar versões com propósitos específicos (Cython, Jython e etc.).

No Índice Tiobe, Python é a 8ª linguagem de programação mais popular, com um grande crescimento nos últimos 10 anos. Desenvolvedores Python ganham em média US$ 75.000,00 anuais (mercado americano) e existem ofertas de emprego quase diárias em sites como PythonJobs.com, muitas delas como home office.
