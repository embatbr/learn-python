Hello World e mais umas coisinhas

Como de praxe, nosso primeiro programa é o já batido Hello World. Vimos na aula anterior que um simples print("Hello World!") escreve na saída Hello World!. Para escrever apenas esta mensagem, seu arquivo hello.py precisa apenas da linha print("Hello World!"). Não é necessário importar algo nem criar um objeto "Printer". A função print() faz parte de um pacote padrão importado automaticamente pelo interpretador. É uma função built-in.

A função print() aceita qualquer quantidade de argumentos. Sua assinatura é print(*objects, sep=' ', end='\n', file=sys.stdout), onde objects são os parametros variáveis a ser escritos (o * indica quantidade variável), sep é o separador dos parametros (comumente um espaço vazio), end é o caractere de fim de linha (por default uma quebra de linha) e file é o arquivo de stream onde a função irá escrever os objects, cujo objeto padrão é a tela do terminal.

Comentários em Python existem apenas de um modo: #. Os comentários são sempre de linha, usando o jogo-da-velha. Para comentar diversas linhas, todas devem iniciar-se com #. Qualquer editor básico possui algum atalho para comentários, geralmente ctrl + /.

Warning!
Caso tente usar caracteres especiais, o interpretador Python pode não entender e escrever coisas estranhas. Para corrigir isto, coloque esta linha como primeira do seu arquivo .py: # -*- coding: utf-8 -*-.


Indentação

Diferentemente das demais linguagens conhecidas (C/C++, Java e etc.), Python não engloba blocos de comandos usando {} (chaves) ou begin e end. Tudo se resume a indentação. O seguinte código, por exemplo, irá compilar: 

print("Hello World!")
print("Aloha Hawaii")
print("Hallo Welt")

Pois todos estão no mesmo nível de indentação. Mas o próximo irá gerar um erro: 

print("Vai...")
print("rodar...")
    print("perfeitamente")

Pois os comandos não estão no mesmo nível. A linguagem é "chatinha" neste ponto por ter como um dos seus princípios a legibilidade do código.


Numerologia

A matemática de Python não é algo de outro mundo. Similar às outras linguagens, em Python + efetua somas, / divide números e > testa se o número à esquerda é maior que o número à direita. A seguir os símbolos matemáticos:

  • + soma
  • - subtração
  • / divisão real
  • // divisão inteira
  • * multiplicação
  • ** exponenciação
  • % resto da divisão (módulo)
  • < menor que
  • > maior que
  • <= menor ou igual que
  • >= maior ou igual que

E seus usos (digite no terminal python3.2 para usar o interpretador interativo): 

>>> 2 + 3
5
>>> 4 - 7
-3
>>> 2 * 4
8
>>> 2 ** 3
8
>>> 4 % 3
1
>>> 2 < 3
True
>>> 2 > 3
False
>>> 2 <= 3
True
>>> 2 >= 2
True
>>> 2 / 3
0.6666666666666666
>>> 2 // 3
0
